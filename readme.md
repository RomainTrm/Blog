# Personal blog

Migration of my old technical blog

## Build reminders

Fetch and pull themes:
```
git fetch
git submodule update --init --recursive
```

Add new post:
```
hugo new content posts/<post-name>.md
```

Builder server with drats for preview:
```
hugo server -D
```
